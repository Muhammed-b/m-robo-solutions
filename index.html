Technical Overview

The Medicine OX Pro is a state-of-the-art autonomous robot designed for hospital environments. It utilizes a 5-channel IR sensor array to navigate complex black-line paths on medical facility floors. The system is governed by an Arduino brain that monitors a Real-Time Clock (RTC) module, triggering precise delivery protocols based on patient-specific medication schedules.

Key Components

  • Arduino Uno R3
  • L298N Motor Driver
  • IR Sensor Array
  • DS3231 RTC Module
  • Li-ion Power Cells

System Architecture

The robot follows a master-slave architecture for motor control and processing. While the primary MCU handles navigation and timing, a secondary controller manages the high-torque servo mechanism responsible for the pill dispensing action, ensuring that medication is dropped accurately into the delivery tray only upon reaching the target destination.

if (isTimeForMeds()) { stopRobot(); dispensePill(); }

Technical Overview

The RFID Matrix is a cloud-integrated attendance tracking solution. Using the high-frequency MFRC522 reader, it scans 13.56MHz passive RFID tags/cards. The ESP32 MCU processes the unique ID (UID) of each card and transmits the data via an encrypted WiFi connection to a Google Sheets database for real-time logging and management.

Key Components

  • ESP32 DevKit V1
  • MFRC522 RFID Module
  • I2C LCD Display
  • 5V Relay Module
  • Active Buzzer

Identity Verification

Beyond simple logging, the system includes a local validation database. When a card is scanned, the status is cross-referenced; authorized personnel are greeted with a green LED strobe and an audible success tone, while unauthorized attempts trigger a high-frequency alarm and an instant push notification to security via IFTTT.

if (rfid.PICC_ReadCardSerial()) { logToCloud(rfid.uid); }

Technical Overview

The POV Visualizer exploits the Persistence of Vision phenomenon to create 3D-like holographic text displays in mid-air. By spinning a single row of high-speed LEDs at over 2000 RPM, and modulating their patterns at microsecond intervals, the human eye perceives a continuous, stable image.

Key Components

  • Arduino Nano
  • Hall Effect Sensor
  • High-Speed LEDs
  • DC Brushless Motor
  • Slip Ring Assembly

Synchronization Logic

The core challenge is timing synchronization. A magnet fixed to the base triggers the Hall Effect sensor on the rotating arm once per revolution. This trigger resets the internal timer, ensuring the text rendering starts at the exact same spatial position every time, regardless of battery-induced fluctuations in motor speed.

// Render Frame based on Hall Effect trigger
if (sensor == LOW) { drawHologram(); }

Technical Overview

Eco-Bin AI is a responsive waste management unit that promotes hygiene through zero-touch interaction. It uses a sonar-based detection algorithm to identify users within 20cm. Upon detection, it triggers a PWM signal to a high-torque servo, smoothly opening the lid for a predefined window before automatic closure.

Key Components

  • ATmega328P MCU
  • HC-SR04 Sonar
  • SG90 Micro Servo
  • 9V Power Pack
  • Status Indicators

Intelligent Timing

The firmware incorporates a 'debounce' logic for ultrasonic readings to prevent false triggers from environmental shifts. If the user remains in range, a 'keep-open' protocol is enacted, ensuring the lid doesn't close prematurely while waste is being deposited.

if (distance < 20) { openLid(); delay(3000); closeLid(); }

Technical Overview

Hydro-Sense is an advanced IR-based touchless water dispensing system. Designed for extreme water conservation, the system uses a high-sensitivity IR proximity sensor paired with a solid-state relay to control a 12V DC water pump. The flow is instantaneous and stops immediately when the vessel is removed.

Key Components

  • Arduino Nano
  • IR Proximity Sensor
  • Optoisolated Relay
  • 12V Diaphragm Pump
  • Inline Water Filter

Hygiene & Efficiency

By removing the physical contact required with traditional taps, Hydro-Sense significantly reduces the spread of germs in shared spaces. The digital control circuit ensures zero 'drip-waste', potentially saving up to 30% more water compared to manual dispensing systems.

if (irSensor == LOW) { relayOn(); } else { relayOff(); }

Technical Overview

The CNC Plotter is a precise 2.5D drawing machine built from repurposed electronics. It interprets standard G-code commands via an Arduino-based interpreter to drive high-precision stepper motors. The system provides a 40mm x 40mm work area perfect for detailed vector art and micro-circuit layout plotting.

Key Components

  • Arduino Uno
  • CNC Shield V3
  • A4988 Stepper Drivers
  • DVD Stepper Motors
  • Micro-Servo Pen Lift

Kinematic Control

Using the Bresenham line algorithm for coordinate translation, the plotter ensures linear motion between points with micro-step precision. The z-axis is managed by a micro-servo that acts as a pen-lift mechanism, controlled by M03/M05 G-code commands.

void processGCode() { moveTo(x, y); penDown(); }
